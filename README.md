# Home Blender Models

This repository contains 3D models for home and interior designs.

## About

This project is a collection of high-quality 3D assets focused on home and interior design. These models are specifically crafted for use in various interactive applications, including virtual reality (VR), augmented reality (AR), video games, and architectural visualizations.

## Contents

- `interior.blend`: Blender file containing the interior design 3D model.
- `Home.mb`: Maya binary file for the home model.

## Usage

These models can be opened in their respective 3D software (Blender for `.blend` files, Autodesk Maya for `.mb` files) for viewing, editing, or integration into other projects such as AR/VR applications or games.
